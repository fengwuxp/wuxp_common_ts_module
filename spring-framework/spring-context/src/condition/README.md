

#### 条件

- 条件类型
```
1：字符串或字符串数组
   (1) SPEL expression ,例如${spring.a.b}，值为真表示通过
   (2) 功能前缀定义，例如：AUTH:['member.add',"member.edit"]:& 表示需要通过用户权限的检查（新增、编辑）
   (3)：如果是数组 则有两种情况
     (3.1):严格模式，必须所有的条件检查都通过才通过  (与计算  &)  ["condition_1","condition_2","&"]  数组的最后一个元素表示计算表达式
     (3.2):宽松模式，有一个条件通过即通过  (或计算  |)           ["condition_1","condition_2","|"]
   

2：布尔值：通过或不通过

3：函数：通过传入 context以及其他的相关参数进行进行计算得到1或2的结果

```
